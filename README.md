GenAI Workflow Onboarding

### 1. Project Description
This repository is a submission for Homework 1 of the GenAI course. It demonstrates a basic workflow using an AI coding assistant (Codex) to generate content, followed by manual human editing, and tracking the entire process using Git and GitHub.

### 2. Git Commands Used
Throughout this project, I used the following Git terminal commands:
* `git clone`: To copy the remote repository to my local machine.
* `git status`: To check the state of my working directory and staging area.
* `git add .`: To stage all changes for the next commit.
* `git commit -m "..."`: To record snapshots of the repository with descriptive messages.
* `git push`: To upload local repository content to GitHub.
* `git pull`: To fetch and integrate remote changes (used to resolve conflicts).
* `git remote set-url`: To update the remote URL with a Personal Access Token (PAT) for authentication.
